The HTML Recipes Project is a beginner-friendly web development exercise focused on building a simple website that showcases a collection of recipes. Using only basic HTML, the project involves creating multiple pages, structuring content with headings, lists, links, and images, and organizing everything into a clear, navigable layout.

By completing this project, I will have demonstrated a solid understanding of fundamental HTML concepts, including page structure, semantic elements, linking between pages, and proper use of text and media. I’ll also gain experience in organizing files and thinking about user navigation, laying an important foundation for more advanced web development skills like CSS styling and JavaScript interactivity
